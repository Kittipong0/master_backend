# Department Management System

โปรเจคนี้เป็น Backend สำหรับเว็บแอปพลิเคชัน (Master web) ที่สร้างด้วย .NET Core Web API, Angular และ SQL Server

## Features

- Manage departments (CRUD)
- Search departments
- Angular Material UI

## Technologies

- Backend: .NET Core Web API (ver.10)
- Database: SQL Server

## Getting Started

1. Clone the repo
2. Run `dotnet run` in backend
